# Scenario Architect v8

シナリオをフローチャート形式で視覚的に管理するためのブラウザベースツールです。  
`index.html` をブラウザで開くだけで動作します（サーバー不要）。

---

## 起動方法

```
index.html をブラウザで直接開く、または GitHub Pages 等で配信する
```

---

## ノードの種類

| 種別 | タイトルバー色 | 用途 |
|------|------|------|
| **TEXT** | 青 | セリフ・地の文・演出など通常のシーン |
| **CHOICE** | 赤 | 分岐ポイント。子 OPTION ノードを自動生成 |
| **OPTION** | 橙 | CHOICE の各分岐先ラベル |

---

## 基本操作

### ノードの作成
- 上部タブバーの **＋ テキスト** / **＋ 選択肢** から追加

### ノードの編集
- ノードをタップ → 下部パネルが開く
- テキストを入力して「完了」

### ノードの接続
1. 接続元をタップ（ハイライト状態）
2. 接続先をタップ → 矢印が引かれる
- 同じ組み合わせを再度タップすると接続解除

### ノードの移動
- ドラッグで自由移動
- CHOICE ノードを移動すると配下の OPTION が追従
- OPTION ノードの個別移動も可能（オフセットが記憶される）

### コンテキストメニュー（長押し）

| 操作 | 説明 |
|------|------|
| ロック切替 | 固定/解除。固定ノードはタップしても編集パネルが開かない |
| コピー | ノードを複製（OPTION は対象外） |
| 削除 | ノードと関連エッジを削除。CHOICE は配下 OPTION も一括削除 |

---

## タブバー（上部）

左右スワイプで隠れているボタンにアクセスできます。

| ボタン | 機能 |
|--------|------|
| ＋ テキスト | TEXT ノードを追加 |
| ＋ 選択肢 | CHOICE ノードを追加（デフォルト 2 分岐） |
| 💾 保存 | JSON ファイルとしてダウンロード |
| 📂 読込 | 保存した JSON を読み込む |
| 🖼 画像 | 現在の表示を PNG でダウンロード |
| 🎯 全体表示 | 全ノードが収まるようビューを自動調整 |
| ↩ 元に戻す | 直前の操作を取り消す（最大 50 段階） |
| ↪ やり直す | 元に戻した操作を再適用（最大 50 段階） |

---

## ミニマップ（右上）

**スクロール / ズーム中と、その終了後 1 秒間のみ表示されます。**  
タブバーと重ならないよう、タブバーの直下に表示されます。

- タップするとその座標にカメラがジャンプ
- 白い矩形が現在の表示範囲
- 色付きドットが各ノード（色はノード種別と同じ）
- タブバー以外の場所をタップすると即座に非表示

スクロール可能な仮想空間は ±4000 × ±3000 に制限されています。

## キーボードショートカット

| キー | 機能 |
|------|------|
| `Delete` / `Backspace` | 選択中のノードを削除 |
| `Ctrl + Z` / `⌘ + Z` | 元に戻す |
| `Ctrl + Y` / `⌘ + Y` | やり直す |
| `Ctrl + Shift + Z` | やり直す（代替） |

---

## 読込時の安全性

読み込んだ JSON は以下を検証してから適用します。

- ファイルサイズ上限 **2 MB**
- ノード数上限 **500**、エッジ数上限 **2000**
- `type` は `text / choice / option` のみ許可
- 文字列から制御文字を除去
- 座標値・分岐数を正当な範囲にクランプ
- 存在しないノード ID を参照するエッジはエラー
- 自己ループはエラー

## ファイルフォーマット（JSON）

```json
{
  "nodes": [
    {
      "id": "text_1234567890",
      "type": "text",
      "rawText": "本文（改行含む）",
      "label": "本文",
      "x": 100, "y": 200,
      "fixed": false
    },
    {
      "id": "choice_111",
      "type": "choice",
      "rawText": "どうする？",
      "label": "どうする？",
      "childIds": ["opt_choice_111_1", "opt_choice_111_2"],
      "childOffsets": {
        "opt_choice_111_1": { "x": 240, "y": -55 },
        "opt_choice_111_2": { "x": 240, "y":  55 }
      },
      "childCount": 2,
      "x": 0, "y": 0,
      "fixed": false
    }
  ],
  "edges": [
    { "from": "text_1234567890", "to": "choice_111" }
  ]
}
```
