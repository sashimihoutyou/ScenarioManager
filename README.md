# Scenario Architect v7

シナリオをフローチャート形式で視覚的に管理するためのブラウザベースツールです。  
`index.html` をブラウザで開くだけで動作します（サーバー不要）。

---

## 起動方法

```
index.html をブラウザで直接開く、または GitHub Pages 等で配信する
```

---

## ノードの種類

| 種別 | タイトルバー色 | 用途 |
|------|------|------|
| **TEXT** | 青 | セリフ・地の文・演出など通常のシーン |
| **CHOICE** | 赤 | 分岐ポイント。子 OPTION ノードを自動生成 |
| **OPTION** | 橙 | CHOICE の各分岐先ラベル |

---

## 基本操作

### ノードの作成
- **+ テキスト** / **+ 選択肢** ボタンで画面中央に追加

### ノードの編集
- ノードをクリック → 下部パネルが開く
- テキストを入力して「完了」

### ノードの接続
1. 接続元をクリック（ハイライト状態）
2. 接続先をクリック → 矢印が引かれる
- 同じ組み合わせをもう一度クリックすると接続解除

### ノードの移動
- ドラッグで自由移動
- CHOICE ノードを移動すると配下の OPTION が追従
- OPTION ノードの個別移動も可能（オフセットが親に記憶される）

### コンテキストメニュー（長押し）
| 操作 | 説明 |
|------|------|
| ロック切替 | ノードを固定/解除。固定ノードはクリックしても編集パネルが開かない |
| コピー | ノードを複製（OPTION は対象外） |
| 削除 | ノードと関連エッジを削除。CHOICE 削除時は配下 OPTION も一括削除 |

## ツールバー

| ボタン | 機能 |
|--------|------|
| 保存 | JSON ファイルとしてダウンロード |
| 読込 | 保存した JSON ファイルを読み込む |
| 画像出力 | 現在の表示を PNG としてダウンロード |
| 全体表示 | 全ノードが収まるようにビューを自動調整 |
| 元に戻す | 直前の操作を取り消す（最大 50 段階） |
| やり直す | 元に戻した操作を再適用（最大 50 段階） |

---

## ミニマップ

画面右上に常時表示されます。

- **白い矩形** = 現在の表示範囲
- **色付きドット** = 各ノード（色はタイプと同じ）
- **クリック** = その位置にビューをアニメーション移動

スクロール可能な範囲は ±4000 × ±3000 の仮想空間に制限されています。

| キー | 機能 |
|------|------|
| `Delete` / `Backspace` | 選択中のノードを削除 |
| `Ctrl + Z` / `⌘ + Z` | 元に戻す |
| `Ctrl + Y` / `⌘ + Y` | やり直す |
| `Ctrl + Shift + Z` | やり直す（代替） |

## ファイルフォーマット（JSON）

```json
{
  "nodes": [
    {
      "id":        "text_1234567890",
      "type":      "text",
      "rawText":   "本文テキスト（改行含む）",
      "label":     "本文テキスト",
      "x":         100,
      "y":         200,
      "fixed":     false
    },
    {
      "id":        "choice_9876543210",
      "type":      "choice",
      "rawText":   "どうする？",
      "label":     "どうする？",
      "childIds":  ["opt_choice_9876543210_1", "opt_choice_9876543210_2"],
      "childOffsets": {
        "opt_choice_9876543210_1": { "x": 240, "y": -55 },
        "opt_choice_9876543210_2": { "x": 240, "y":  55 }
      },
      "childCount": 2,
      "x": 0, "y": 0, "fixed": false
    }
  ],
  "edges": [
    { "from": "text_1234567890", "to": "choice_9876543210" }
  ]
}
```

### 主なノードプロパティ

| プロパティ | 型 | 説明 |
|---|---|---|
| `id` | string | 一意識別子 |
| `type` | `"text"` \| `"choice"` \| `"option"` | ノード種別 |
| `rawText` | string | 編集パネルで入力したテキスト（改行含む） |
| `label` | string | 表示用ラベル（rawText の先頭 4 行） |
| `x`, `y` | number | キャンバス座標（範囲: ±4000 / ±3000） |
| `fixed` | boolean | ロック状態 |
| `childIds` | string[] | CHOICE のみ。配下 OPTION の ID 一覧 |
| `childOffsets` | object | CHOICE のみ。子ノードの親からの相対座標 |
| `childCount` | number | CHOICE のみ。分岐数 (1〜9) |

---

## 読込時の安全性

読み込んだ JSON は以下の検証・サニタイズを経てから適用されます。

- ファイルサイズ上限: **2 MB**
- ノード数上限: **500**、エッジ数上限: **2000**
- `type` は `text / choice / option` のみ許可
- 文字列から制御文字を除去・長さを制限
- 座標値・分岐数を正当な範囲内にクランプ
- 存在しないノード ID を参照するエッジはエラー
- 自己ループエッジはエラー
