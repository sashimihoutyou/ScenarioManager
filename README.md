# Scenario Architect

シナリオをフローチャート形式で視覚的に管理するためのブラウザベースツールです。  
`index.html` を開くだけで動作します（サーバー不要）。

---

## 起動方法

```
index.html をブラウザで開く
```

---

## ノードの種類

| 種別 | 色 | 用途 |
|------|------|------|
| **テキスト** (TEXT) | 青 | セリフ・地の文・演出など、通常のシーン |
| **選択肢** (CHOICE) | 赤 | 分岐ポイント。子 OPTION ノードを自動生成 |
| **選択肢子** (OPTION) | 橙 | CHOICE の各分岐先ラベル。直接接続先を繋ぐ |

---

## 基本操作

### ノードの作成
- **＋ テキスト** ボタン → TEXT ノードを画面中央に追加
- **＋ 選択肢** ボタン → CHOICE ノードを追加（デフォルト2分岐）

### ノードの編集
- ノードをクリック → 下部パネルが開く
- テキストを入力して「完了」

### ノードの接続
1. 接続元ノードをクリック（ハイライト状態）
2. 接続先ノードをクリック
→ 矢印が引かれる。同じ組み合わせをもう一度クリックすると接続解除

### ノードの移動
- ドラッグで自由移動
- CHOICE ノードを移動すると、配下の OPTION ノードが追従
- OPTION ノード個別の移動も可能（オフセットが更新される）

### コンテキストメニュー（長押し）
| 操作 | 説明 |
|------|------|
| 🔒 ロック切替 | ノードを固定/固定解除。固定ノードはクリックしても編集パネルが開かない |
| 📄 コピー | ノードを複製（OPTION ノードは対象外） |
| 🗑 削除 | ノードと関連エッジを削除。CHOICE 削除時は配下 OPTION も一括削除 |

---

## ツールバー

| ボタン | 機能 |
|--------|------|
| 💾 保存 | JSON ファイルとしてダウンロード（`scenario_YYYY-MM-DD.json`） |
| 📂 読込 | 保存した JSON ファイルを読み込む |
| 🖼 画像出力 | 現在の表示を PNG としてダウンロード |
| ↩ 元に戻す | 直前の操作を取り消す（最大 30 段階） |

---

## キーボードショートカット

| キー | 機能 |
|------|------|
| `Delete` / `Backspace` | 選択中のノードを削除 |
| `Ctrl + Z` / `⌘ + Z` | 元に戻す |

## ファイルフォーマット（JSON）

保存ファイルは以下の構造を持つ JSON です。

```json
{
  "nodes": [
    {
      "id":        "text_1234567890",
      "type":      "text",
      "rawText":   "本文テキスト",
      "label":     "本文テキスト",
      "x":         100,
      "y":         200,
      "fixed":     false
    },
    {
      "id":        "choice_9876543210",
      "type":      "choice",
      "rawText":   "選択してください",
      "label":     "選択してください",
      "childIds":  ["opt_choice_9876543210_1", "opt_choice_9876543210_2"],
      "childOffsets": {
        "opt_choice_9876543210_1": { "x": 240, "y": -55 },
        "opt_choice_9876543210_2": { "x": 240, "y":  55 }
      },
      "childCount": 2,
      "x":          0,
      "y":          0,
      "fixed":      false
    }
  ],
  "edges": [
    { "from": "text_1234567890", "to": "choice_9876543210" }
  ]
}
```

### ノードプロパティ

| プロパティ | 型 | 説明 |
|---|---|---|
| `id` | string | 一意識別子 |
| `type` | `"text"` \| `"choice"` \| `"option"` | ノード種別 |
| `rawText` | string | 編集パネルでの入力テキスト（改行含む） |
| `label` | string | vis-network 表示用ラベル（rawText の先頭4行） |
| `x`, `y` | number | キャンバス上の座標 |
| `fixed` | boolean | ロック状態 |
| `childIds` | string[] | CHOICE ノードのみ。配下 OPTION の ID 一覧 |
| `childOffsets` | object | CHOICE ノードのみ。子ノードの相対座標 |
| `childCount` | number | CHOICE ノードのみ。分岐数 |

## ブラウザ要件

- Chrome / Edge / Firefox の最新版
- Safari 16 以降（`roundRect` API 対応）
- ローカルファイル読込のため、ブラウザのセキュリティ設定で `file://` プロトコルが許可されている必要があります
